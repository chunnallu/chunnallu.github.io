包含两个例子，你可以直接点击其中的index.html打开。如果要运行源代码，需要安装相应环境（node、webpack、babel）。

这两个例子主要用来说明[js自定义事件的使用](http://blog.csdn.net/ruangong1203/article/details/52474452)

操作：点击A或双击A
